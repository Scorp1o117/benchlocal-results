✅ 该夸的要夸

  • 工具调用教科书级 — A-D 维度全满分，精准选对 15 个工具，参数格式零失误

  • Agent 有亮点 — 浏览器登录导出、审批守卫、技能创建全满分

❌ 该吐槽的绝不嘴软

  • 🏆 Trap 题两连跪 — 用户说有 bug 就急着修，198B 模型连 format! 不夺走所有权都想不明白
  • 🐌 Thinking 关不掉 — 简单题想 16 分钟，等好久然后给错误答案
  • 🤦 Agent 执行力不稳 — 问完"删哪个"没等回答就动手，JSON key 差一个驼峰

💡 给老师的建议

  • 双模型策略 — Qwen3 日常 + Step-3.7 复杂任务
  • Prompt 加固 — "先独立验证，再决定是否修复"
  • parallel 改 2-3 — 避免 fetch failed
